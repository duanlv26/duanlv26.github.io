DOM - Document Object Moudel

发表于 2018-01-15

文档：DOM中的”D”

如果没有 document(文档) ，DOM 也就无从说起。

对象：DOM中的”O”

JavaScript中的对象可以分为三种类型：

  • 用户定义对象(user-defined object)：由我们自行创建的对象。
  • 内建对象(native object)：内建在JavaScript中的对象，如Array、Math等。
  • 宿主对象(host object)：由浏览器提供的对象。

模型：DOM中的”M”

可以把网页的结构以’树’的形式显示出来。

<DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8">
        <title>deom</title>
    </head>
    <body>
        <h1>This is h1</h1>
        <p>This is P</p>    
        <ul>
            <li></li>
            <li></li>
            <li></li>
        </ul>
    </body>

节点(node)

节点是网络术语，它表示网络中的一个链接点。一个网络就是由一些节点构成的集合。

在DOM中有许多不同类型的节点，下面介绍了其中的三种:元素节点、文本节点和属性节点。

<p title = "a gentle reminder"> Do not forget to buy this stuff.</p>
其中<p></p>是元素节点，<title="a gentle reminder">是属性节点，Do not forget to buy this stuff.是文本节点。

元素节点(element node)

在之前的HTML代码中，使用了 <body> <p> <ul> 之类的元素。
如果web上的文档是一座大厦，元素就是构建大厦的砖块，这些元素的布局构成了文档的结构。

文本节点(text node)

如果一份文档完全由一些空白元素构成，它将有一个结构，但是没有内容。
<p>元素，它就是一个文本节点。

在HTML中，文本节点总是被包含在元素节点中。

属性节点(object node)

属性节点用来对元素做出更具体的描述。例如，某个元素中具有title属性，我们可以利用这个属性对包含在元素里的东西做出准确的描述：<p title="abcdefg">Hello!</p>
在DOM中，’title="abcdefg‘是一个属性节点。
属性节点总是被包含在元素节点中。

CSS

我们可以通过CSS告诉浏览器如何显示文档的内容。

类似JavaScript脚本，对样式的声明既可以嵌在文档<head>部分，也可以放在另一个样式表文件里。

p{
    color:red;    font-size:15px;
}

获取元素

有三种DOM方法获取元素节点，分别是通过元素ID、通过标签名字和通过类名字来获取。

1.getElementById

  DOM提供了名为getElementById的方法、这个方法将返回一个与那个有着给定id属性值得元素节点对应的对象。

  它是 document对象的特有函数。getElementById方法只有一个参数：你想获得的那个元素的id属性的值。

document.getElementById(id);

下面是个例子：

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Document</title>
</head>
<body>
    <h1>What you buy</h1>
    <p title=" a gentle reminder">Do not forget to buy this stuff.</p>    
    <ul id="purchases">
            <li>A</li>
            <li class="sale">B</li>
            <li class="sale important">C</li>
    </ul>

<script type="text/javascript">
    alert(typeof document.getElementById("purchases"));
    //会弹出一个内容是object的对话框，它向你报告document.getElementById的类型。(它是一个对象)
</script>

</body>
</html>

这个调用将返回一个对象，这个对象对应着document对象里的独一无二的元素，那个元素的HTML id 属性是 purchases。

  文档的每一个元素都是一个对象，利用DOM方法能得到任意一个对象。

2.getElementsByTagName

  getElementsByTagName方法返回一个对象数组，每个对象分别对应着文档里有着给定标签的一个元素。

document.getElementsByTagName("li")
//这个调用将返回一个对象数组。

alert(document.getElementsByTagName("li").length)
//弹出这份文档中列表项元素的个数。(3个)

//可以用for循环和typeof操作符验证
for (var i=0; i < document.getElementsByTagName("li").length; i++){
    alert(typeof document.getElementsByTagName("li")[i]);
}
//可以改善代码，如下：
var items = document.getElementsByTagName("li")；
for (var = 0 ; i < items.length; i++){
    alert(typeof items[i]);
}
//显示三个对话框，全部是object。

也允许getElementByTagName把一个通配符(“ * “)作为参数，这意味着文档里的每一个元素都在函数返回的数组里具有地位。

alert(document.getElementsByName("*").length);

可以把getElementById和getElementsByName结合使用：

var shopping = document.getElementById("purchases");
var items = shopping.getElementByTagName(" * ");
//item数组只包含id为 purchases(id="purchase")的 li 里的元素

3.getElementsByClassName

HTML5 DOM中新增加了这个方法。这个方法使我们可以通过class属性中的类名称来访问元素。与getElementsByTagName相似，也只接受一个参数。

getElementsByClassName("class")；    //只接受类名
//.....也可以接受多个类名("class1 class2")

例

alert(document.getElementsByClassName("important sale").length);
//警告框中显示1，表示只有一个元素匹配，因为只有一个元素同时带有"important"和"sale"两个类名。

获取和设置属性

  我们已经有三种获取特定元素的方法了，得到元素之后，我们就可以设法获取他的各个属性。我们有两种方法：getAttribute和setAttribute。

1.getAttribute

getAttribute是一个函数。它只有一个参数(你打算查询的属性的名字)：

object.getAttribute("...")

getAttribute不属于document对象。它只能通过元素节点对象调用。我们可以和getElementsByTagName方法合用。

//获取每个<p>元素的title属性
var paras = document.getElementsByTagName("p");
for(var i=0; i<paras.length; i++){
    alert(paras[i].getAttribute("title"));
}

//会弹出内容是"a gentle reminder"的对话框

2.setAttribute

setAttribute是用来改变信息的。和get…te一样，也只能用于节点。

object.setAttribute("...","value")

//得到id是purchase的元素，然后把这个元素的title属性设置为a list of goods
var shopping = document.ElementById("purchases");
shopping.setAttribute("title","a list of goods");
