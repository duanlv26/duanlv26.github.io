DOM - Document Object Moudel

发表于 2018-01-15

文档：DOM中的”D”

如果没有 document(文档) ，DOM 也就无从说起。

对象：DOM中的”O”

JavaScript中的对象可以分为三种类型：

  • 用户定义对象(user-defined object)：由我们自行创建的对象。
  • 内建对象(native object)：内建在JavaScript中的对象，如Array、Math等。
  • 宿主对象(host object)：由浏览器提供的对象。

模型：DOM中的”M”

可以把网页的结构以’树’的形式显示出来。

<DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8">
        <title>deom</title>
    </head>
    <body>
        <h1>This is h1</h1>
        <p>This is P</p>    
        <ul>
            <li></li>
            <li></li>
            <li></li>
        </ul>
    </body>

节点(node)

节点是网络术语，它表示网络中的一个链接点。一个网络就是由一些节点构成的集合。

在DOM中有许多不同类型的节点，下面介绍了其中的三种:元素节点、文本节点和属性节点。

<p title = "a gentle reminder"> Do not forget to buy this stuff.</p>
其中<p></p>是元素节点，<title="a gentle reminder">是属性节点，Do not forget to buy this stuff.是文本节点。

元素节点(element node)

在之前的HTML代码中，使用了 <body> <p> <ul> 之类的元素。
如果web上的文档是一座大厦，元素就是构建大厦的砖块，这些元素的布局构成了文档的结构。

文本节点(text node)

如果一份文档完全由一些空白元素构成，它将有一个结构，但是没有内容。
<p>元素，它就是一个文本节点。

在HTML中，文本节点总是被包含在元素节点中。

属性节点(object node)

属性节点用来对元素做出更具体的描述。例如，某个元素中具有title属性，我们可以利用这个属性对包含在元素里的东西做出准确的描述：<p title="abcdefg">Hello!</p>
在DOM中，’title="abcdefg‘是一个属性节点。
属性节点总是被包含在元素节点中。

CSS

我们可以通过CSS告诉浏览器如何显示文档的内容。

类似JavaScript脚本，对样式的声明既可以嵌在文档<head>部分，也可以放在另一个样式表文件里。

p{
    color:red;    font-size:15px;
}

获取元素

有三种DOM方法获取元素节点，分别是通过元素ID、通过标签名字和通过类名字来获取。

1.getElementById

  DOM提供了名为getElementById的方法、这个方法将返回一个与那个有着给定id属性值得元素节点对应的对象。

  它是 document对象的特有函数。getElementById方法只有一个参数：你想获得的那个元素的id属性的值。

document.getElementById(id);

下面是个例子：

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Document</title>
</head>
<body>
    <h1>What you buy</h1>
    <p title=" a gentle reminder">Do not forget to buy this stuff.</p>    
    <ul id="purchases">
            <li>A</li>
            <li class="sale">B</li>
            <li class="sale important">C</li>
    </ul>

<script type="text/javascript">
    alert(typeof document.getElementById("purchases"));
    //会弹出一个内容是object的对话框，它向你报告document.getElementById的类型。(它是一个对象)
</script>

</body>
</html>

这个调用将返回一个对象，这个对象对应着document对象里的独一无二的元素，那个元素的HTML id 属性是 purchases。

  文档的每一个元素都是一个对象，利用DOM方法能得到任意一个对象。

2.getElementsByTagName

  getElementsByTagName方法返回一个对象数组，每个对象分别对应着文档里有着给定标签的一个元素。

document.getElementsByTagName("li")
//这个调用将返回一个对象数组。

alert(document.getElementsByTagName("li").length)
//弹出这份文档中列表项元素的个数。(3个)

//可以用for循环和typeof操作符验证
for (var i=0; i < document.getElementsByTagName("li").length; i++){
    alert(typeof document.getElementsByTagName("li")[i]);
}
//可以改善代码，如下：
var items = document.getElementsByTagName("li")；
for (var = 0 ; i < items.length; i++){
    alert(typeof items[i]);
}
//显示三个对话框，全部是object。

也允许getElementByTagName把一个通配符(“ * “)作为参数，这意味着文档里的每一个元素都在函数返回的数组里具有地位。

alert(document.getElementsByName("*").length);

可以把getElementById和getElementsByName结合使用：

var shopping = document.getElementById("purchases");
var items = shopping.getElementByTagName(" * ");
//item数组只包含id为 purchases(id="purchase")的 li 里的元素

3.getElementsByClassName

HTML5 DOM中新增加了这个方法。这个方法使我们可以通过class属性中的类名称来访问元素。与getElementsByTagName相似，也只接受一个参数。

getElementsByClassName("class")；    //只接受类名
//.....也可以接受多个类名("class1 class2")

例

alert(document.getElementsByClassName("important sale").length);
//警告框中显示1，表示只有一个元素匹配，因为只有一个元素同时带有"important"和"sale"两个类名。

获取和设置属性

  我们已经有三种获取特定元素的方法了，得到元素之后，我们就可以设法获取他的各个属性。我们有两种方法：getAttribute和setAttribute。

1.getAttribute

getAttribute是一个函数。它只有一个参数(你打算查询的属性的名字)：

object.getAttribute("...")

getAttribute不属于document对象。它只能通过元素节点对象调用。我们可以和getElementsByTagName方法合用。

//获取每个<p>元素的title属性
var paras = document.getElementsByTagName("p");
for(var i=0; i<paras.length; i++){
    alert(paras[i].getAttribute("title"));
}

//会弹出内容是"a gentle reminder"的对话框

2.setAttribute

setAttribute是用来改变信息的。和get…te一样，也只能用于节点。

object.setAttribute("...","value")

//得到id是purchase的元素，然后把这个元素的title属性设置为a list of goods
var shopping = document.ElementById("purchases");
shopping.setAttribute("title","a list of goods");

JavaScript-对象

发表于 2018-01-13

对象(object)是数据类型，对象是自包含的数据集合，包含在对象里的数据可以通过两种形式访问：属性(property)和方法(method)。

属性是隶属于某个特定对象的变量。

方法是只有某个特定对象才能调用的函数。

对象就是由一些属性和方法组合在一起而构成的一个数据实体。

Object.property
Object.method()       //在JavaScript中，属性和方法都使用' . '来访问。

假设以一个人为对象，我们要使用 mood 和 age 等变量来储存如 “心情”和“年龄”之类的值。

Person.mood
Person.age            //mood和age是Person的属性

假如 Person 对象还关联着如 walk() 和 sleep() 之类的函数，这些函数就是这个对象的方法。

Person.walk()
Person.sleep()        //walk和sleep是Person的方法

把这些属性和方法全部结合在一起，就得到了一个 Person 对象。

但是不可能只有一个person，虽然都是person对象，但是可能具有不同的属性，我们可以像下面这样利用Person对象的属性来检索关于 wangwei 的信息。

var wangwei = new Person ;

//创建了一个Person对象的新实例。

wangwei.age
wangwei.mood    //我们可以利用Person对象的属性检索wangwei的信息。

内建对象

数组其实就是一种内建对象。
当我们使用new关键字去初始化一个数组时，其实是在创建一个Array对象的新实例。

var word = new Array();

当需要了解某个数组有多少个元素时，利用Array对象的length属性获取信息：

beatles.length;

宿主对象

除了内建对象，还可以在JS脚本里使用一些已经预先定义好的其他对象。 这些对象不是由JS语言本身，而是由它的运行环境提供的。具体到Web应用，这个环境就是浏览器。由浏览器提供的预定义对象被称为宿主对象(host object)。

JavaScript-函数

发表于 2018-01-13

如果要多次使用同一段代码，可以把他们封装成一个函数。


定义一个函数的语法：

function name(arguments){        //arguments类数组对象
    statements;                //statements 声明；语句；    
}

如下是需要传递两个参数的函数。

function add( num1 , num2 ){
    var total = num1 + num2 ;
    alert(total);
}


add( 5 , 10 );    //弹出内容为 ' 15 ' 的对话框。

变量的作用域

变量既可以是全局的，也可以是局部的。

全局变量(global variable)

可以在脚本的任何位置被引用，全局变量的作用域是整个脚本。

局部变量(local variable)

只存在于声明它的那个函数的内部，在外部是无法引用它的。局部变量的作用仅限于某个特定的函数。

可以用 var 关键字来明确地为函数变量设定作用。

如果在某个函数使用了var，那个变量就被视为一个局部变量，它只存在于这个函数的上下文中；反之，如果没有使用var，那个变量就被视为一个全局变量，如果脚本里已经存在一个与之同名的全局变量，这个函数就会改变那个全局变量的值。

如下：

function square(num){            //定义一个名为square的方法
    total = num * num ;
    return total ;
}
var total = 50 ;
var number = square(20);    //导致了全局变量total的值发生了变化。
alert(total);    // 400 .

全局变量total的值变成了400。我们的本意是让square()函数只把它计
算出来的平方值返回给number，但因为没有把这个函数内部的total变量
明确的生命为局部变量，这个函数把名字同样是total的全局变量的值也改
变了。

我们可以把函数改写成：

function square(num){
    var total = num * num ;
    return total ;
}

这样再怎么调用square()，也不会影响全局变量total了。

JavaScript 字符串

发表于 2018-01-10

查看字符串长度：

console.log('Hello world!'.length);     // 12
-----------------------------------------------
var str = 'Hello world!' ;
console.log('str'.length);        // 12 
-----------------------------------------------
var str = 'Hello world!\n' ;    // 13 
                [ \n 代表回车，转义符算作一个字符]
-----------------------------------------------
var len = 'Hello world!'.lengh ;
console,log( len ) ;            // 12

使用 [ ] 操作符取单个字符：

字符串 ABCDE中 a b c d e 是它的成员，
成员在字符串的位置叫索引。

JavaScript中索引从0开始。

成员    | a | b | c | d | e
--------------------------
索引    | 0 | 1 | 2 | 3 | 4

在操作字符串的时候，如果我们想要得到一个字符串中的单个的字符，可以在字符串后使用[]操作符把想要获取的成员的索引包裹住，例如：
console.log(‘abcde’[0]); // a

var str = 'abcde' ;
console.log('str[1]');        // b

[ ] 的内部还能是表达式，以方便我们在编程的时候动态的获取单个成员，例如：

var str = 'ABCDE' ;
console.log(str[ 1 + 2 ]);    // D

var a = 2 , b = 2 ;
console.log(str[ a + b ]);    // E

console.log(str[ str.lengh - 1 ]);    // E

【 lengh - 1 恰好是字符串最后一个成员的索引】

字符串切片(获取字符串中的一部分内容)：

JavaScript 提供了slice方法让我们对字符串进行切片。slice方法的用法如下伪代码描述。

字符串.slice(开始的索引, 结束的索引);

例如下列表格标注出了字符串ABCDEFG各个成员和索引。

成员    | A | B | C | D | E | F | G
----------------------------------
索引    | 0 | 1 | 2 | 3 | 4 | 5 | 6

var str = 'ABCDEFG' ;
var part1 = str.slice(0,3); //获取索引 0 开始到 3 之前的内容。
consloe.log(part1);        // ABC

在slice方法中我们可以省略第二个参数「结束的索引」，那么此时它的行为就变成，获取从「开始的索引」开始一直到字符串结束之前的内容，例如下列代码：

var str = 'ABCDEFG' ;
var part1 = str.slice(3);    //获取从索引 3 一直到字符串结束之前的内容
console.log(part1);        // DEFG

获取成员的索引：

字符串的 indexOf 方法可以获取其中成员的索引。

实例：

成员    | A | B | C | D | E | F | G
----------------------------------
索引    | 0 | 1 | 2 | 3 | 4 | 5 | 6

var str = 'ABCDEFG' ;
var index1 = str.indexOf('B');
console.log(index1);    // 1

indexOf的参数除了可以是单个的字符，还可以是一个字符串，例如下列代码中从字符串’满脑子都是孩子哭了笑了’中获取，’孩子’和’哭了笑了’的索引：

var str = 'ABCDEFG';
var index1 = str.indexOf('DEF');
console.log(index1);     // 3

如果找不到成员，indexOf就会返回-1，例如：

var str = 'ABCDEFG';
var index = str.indexOf('404');
console.log(index);     // -1

变量

  • 变量是存储信息的容器。
  • 必须以字母开头。
  • 变量也能以 $ 和 _ 符号开头（不推荐）
  • 变量名称对大小写敏感

实例

var x=2;        // x 的值为 2
var y=3;        // y 的值为 3
var z=x+y;

JavaScript 变量还能保存其他数据类型，比如文本值 (name=”Bill Gates”)。
